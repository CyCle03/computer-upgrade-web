# [SCA] 컴퓨터 강화하기 V1.2.9 — 원본 유즈맵 참고 문서

이 문서는 원본 유즈맵 **`[SCA]컴퓨터강화하기 V1.2.9.scx`** 를 추출·분석한 결과를 정리한 것입니다.  
웹 복원판 구현 시 원작 규칙을 맞추기 위한 기준 문서로 사용합니다.

> **분석 방법:** SCX(MPQ) 아카이브 추출 → CP949 한글 문자열·수치 테이블 파싱  
> **주의:** SCA/EUD 보호 맵이라 트리거/EUD 로직 전체는 공개되지 않으며, UI 문자열·가이드 수치·상점 표기를 기준으로 작성했습니다.

### 1.1 컴퓨터 구성

CPU, 쿨러, 메인보드, RAM, GPU, HDD/SSD(M.2 NVMe)로 구성됩니다.

- **강화 가능:** CPU, RAM, GPU, 쿨러, 드라이브(HDD/SSD/NVMe)
- **완제품 구매:** 메인보드 (소켓·DDR 규격 선택)
- **Intel CPU:** 1~14강 (Core i5 시리즈)
- **AMD CPU:** 1~6강 (Ryzen™ 5 시리즈)
- **GPU:** 1~10강 (GeForce GT ~ RTX 4090)
- **RAM:** 1~13강 (DDR3 ~ DDR5)
- **드라이브:** HDD 1~4강, M.2 NVMe 1~4강

## 2. 재화 체계

원작에는 **세 종류의 재화 개념**이 있습니다.

```
미네랄(원)  ──[10,000 : 1]──▶  Normal 코인(가스 UI)  ──▶  일반 상점(부품·강화)
                                      │
                                   환생 시 초기화

SCA 코인  ──▶  SCA 센터(영구 성장)  ──▶  환생해도 유지 (SCA 런처 저장)
```

### 2.1 미네랄 (원)

- 사냥터·채굴·작업 등으로 획득하는 **기본 자원**
- UI 표시 단위: `1원 단위`, `1만원 단위`
- 일정량이 쌓이면 **Normal 코인으로 자동 환전** (v1.1.1 패치노트: 「게임 중 미네랄이 코인으로 환전」)

### 2.2 Normal 코인 (인게임 코인)

| 항목 | 내용 |
|------|------|
| **표시** | 스타 UI **가스** 칸 |
| **환전** | **미네랄 10,000 = Normal 코인 1** (UI `1코인 단위` = `1만원 단위` 구조) |
| **용도** | 일반 상점에서 CPU/GPU/RAM/쿨러/드라이브 **구매·강화** |
| **초기화** | **환생 시** 부품·다운로드·돈(미네랄·Normal 코인) 전부 리셋 |

### 2.3 SCA 코인 (클라우드 영구 재화)

| 항목 | 내용 |
|------|------|
| **저장** | SCA 런처 서버 (웹 복원판: PostgreSQL `permanent_currencies`) |
| **획득** | 환생 보상, 파티 사냥터 단계 보상 등 |
| **용도** | SCA 센터 영구 성장 (아래 §4 참고) |
| **초기화** | **환생해도 유지·누적** |

## 4. SCA 센터 · SCA 코인 상점

### 4.1 SP(SCA Point) 상점

맵 내부 `SP : N` 표기와 v1.0.3~v1.1.5 패치노트를 대조한 추정 매핑:

| SP 비용 | 추정 효과 |
|---------|-----------|
| **500** | 시작/환생 미네랄 +500 |
| **800** | 환생 미네랄 최대 +200 |
| **5,000** | 환생 미네랄 최대 +2,000 |
| **8,000** | 환생 미네랄 최대 +7,500 |
| **12,000** | (중급 성장 항목) |
| **25,000** | (고급 성장 항목) |
| **30,000** | (고급 성장 항목) |
| **40,000** | (최고급 — GPU 등급·확률 등) |

v1.1.4 기준 SCA 상점 대표 항목 (패치노트):

- **환생 미네랄 최대** 증가 (최대 30,000까지)
- **사냥터 수입** +1% (누적)
- **게임 배속** +1프레임 (최대 10)
- **강화 확률** +0.1% (최대 1.0%)
- **GPU 등급** 증가
- **Intel CPU 11강** (4코어) 구매
- **외형 / 무기 커스텀 센터**

### 4.2 SCA 코인 직접 구매 표기

맵 UI에 확인된 SCA 코인 가격 단위:

`2` · `20` · `200` · `500` · `2,000` · `5,000` · `7,500` · `40,000` 코인

### 4.3 쿠폰

| 쿠폰 | 설명 |
|------|------|
| **배속쿠폰** | 클릭 시 1장 사용, 방 배속 증가 |
| **수입쿠폰** | 클릭 시 1장 사용, 사냥 수입 증가 |

## 6. 사냥터 · 수입

### 6.1 파티 사냥터

**원(미네랄)** 과 **SCA 코인**을 단계별로 동시에 지급합니다.

**1번 사냥터 라인 (원)**

`30원` → `300원` → `3,500원` → `50,000원` → `750,000원`

**1번 사냥터 라인 (SCA 코인)**

`2코인` → `20코인` → `200코인` → `2,000코인` → `5,000코인` → `40,000코인`

**2번 사냥터 라인 (원)**

`1원` → `100원` → `2,500원` → `30,000원` → `250,000원` → `5,000,000원`

**2번 사냥터 라인 (SCA 코인)**

`20코인` → `500코인` → `7,500코인`

### 6.2 오버클럭 사냥터

- DDR4 / DDR5 **오버클럭 파츠** 획득
- 오버클럭 연구소 레벨, 자동 사용 ON/OFF 설정

### 6.3 작업 사냥터

건물 파괴 기반 수입. RAM 용량에 따라 배치 가능 유닛 수 제한.
